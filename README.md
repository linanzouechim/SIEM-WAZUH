# SIEM-WAZUH


Ce projet a pour objectif d’installer, configurer et exploiter Wazuh dans un environnement d’expérimentation afin de simuler un centre de surveillance de sécurité (SOC).

L’environnement comprend :

Un serveur Wazuh

Deux machines clientes (agents) de familles de systèmes d’exploitation différentes :

Windows

Linux / macOS

L’objectif est de :

Déployer Wazuh

Connecter les agents

Détecter des événements de sécurité

Analyser les alertes

Produire un rapport de détection
